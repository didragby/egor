Главная HTML CSS JS
JavaScript является объектно-ориентированным языком, но используемое в языке прототипирование[38][39] обуславливает отличия в работе с объектами по сравнению с традиционными класс-ориентированными языками. Кроме того, JavaScript имеет ряд свойств, присущих функциональным языкам — функции как объекты первого класса, объекты как списки, карринг, анонимные функции, замыкания[40] — что придаёт языку дополнительную гибкость. Несмотря на схожий с Си синтаксис, JavaScript по сравнению с языком Си имеет коренные отличия: объекты с возможностью интроспекции; функции как объекты первого класса; автоматическое приведение типов; автоматическая сборка мусора; анонимные функции. В языке отсутствуют такие полезные вещи[41], как: стандартная библиотека: в частности, отсутствует интерфейс программирования приложений по работе с файловой системой, управлению потоками ввода-вывода, базовых типов для бинарных данных; стандартные интерфейсы к веб-серверам и базам данных; система управления пакетами[~ 2], которая бы отслеживала зависимости и автоматически устанавливала их'
